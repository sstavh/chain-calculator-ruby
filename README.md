# Advanced Chain Calculator (Ruby)

This project is a command-line calculator written in Ruby.

It supports both **chain calculations** and **full expression parsing with parentheses**, and is implemented using **object-oriented programming (OOP)**.

## ✨ Features

- Chain calculations (step-by-step input)
- Expression evaluation with parentheses
- Binary operations: `+`, `-`, `*`, `/`, `mod`, `pow`
- Unary operations: `--`, `sqrt`, `sin`, `cos`, `tan`, `ctan`, `exp`, `ln`, `!`
- Memory support:
  - `mw` — memory write
  - `mr` — memory read
- Calculation history (`history`)
- Input validation and error handling
- Supports both:
  - interactive input
  - full expressions (e.g. `2 + 3 * (4 + 1)`)

## ▶️ How to run

```bash
ruby calculator.rb
